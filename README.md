# Cutter::Knife

Nokogiri Perser Checker - http://cutter-knife.herokuapp.com

## Installation

Add this line to your application's Gemfile:

    $ git clone http://github.com/naoto/cutter-knife.git

And then execute:

    $ cd cutter-knife
    $ bundle install
    $ bundle exec ruby web.rb

## Usage

Path to CSS Path or xpath
